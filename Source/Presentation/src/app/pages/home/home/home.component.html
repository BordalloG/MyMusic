MyMusic

Seja bem vindo!

O que é o MyMusic:

Gerenciador de playlits que permite que você adicione músicas de forma simples e fácil, permitindo que as edite ou exclua de sua lista. Além disso o sistema conta com uma Integração do Spotify que permite importar as músicas de uma playlist passando a url da mesma.

Como o MyMusic foi desenvolvido:

MyMusic foi desenvolvido usando .net core para criar uma API, e seu front-end fui utilizado Angular. Você pode chegar o código font clicando aqui.
